# 한국어 + 한영 혼용 동의어 사전 (synonym_graph 용)
#
# 포맷:
#   A, B, C          : 양방향(상호 치환) — 어떤 토큰으로 검색해도 나머지가 매칭
#   A, B => X        : 단방향(치환) — A/B 로 검색하면 X 로 확장
#
# 주의:
#   1) 각 항목은 한 줄. 공백이 포함된 다단어 동의어는 synonym_graph 가 처리함.
#   2) nori 가 "mixed" 모드로 분해한 토큰과 매칭되어야 하므로,
#      여기 등재한 원어는 가급적 nori 의 분해 결과와 일치시킬 것.
#   3) 작은 사전으로 시작하고, 운영하면서 오검색 로그를 보며 확장.
